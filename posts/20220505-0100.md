<h2>70. Climbing Stairs</h2>
<pre><code class="language-python">
def climbStairs(n):
    if n < 3:
        return n
    x, y = 1, 2
    for _ in range(n - 2):
        ans = x + y
        x = y
        y = ans
    return ans
</code></pre>

<h2>88. Merge Sorted Array</h2>

<pre><code class="language-python">
def merge(nums1, m, nums2, n):
    while len(nums1) > m:
        nums1.pop(m)
    while len(nums2) > n:
        nums2.pop(n)
    for i in nums2[:]:
        nums1.append(i)
    nums1.sort()
